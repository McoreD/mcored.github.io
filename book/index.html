Workshop booking

Schedule capacity-limited workshop sessions for a series, share a link, and track who’s still to say Yes.

How it works

  1. Admin opens the secret admin URL.
  2. Create a series, import names from CSV, set a workshop date/time/capacity.
  3. Copy the workshop link into Outlook yourself.
  4. Guests pick their name and Yes/No. Repeat sessions until everyone has said Yes.
